**x86 固件更新 方式：**

1. 在**OP后台 系统--更新固件 点击 手动更新** 稍等几分钟 路由重启即可
2. **命令行** 或 **SSH 链接** OP **执行以下命令** 完成固件更新
- 执行 **`bash /bin/AutoUpdate.sh`** 保留配置更新
- 执行 **`bash /bin/AutoUpdate.sh -n`** 不保留配置更新

**注意：升级不会保留原有自己安装的 app ，还需升级后自行按需安装**
